🔧 Lenguajes de Programación

🐍 Python

Python es un lenguaje de alto nivel, fácil de leer y escribir. Se utiliza ampliamente en ciencia de datos, inteligencia artificial, automatización y desarrollo web.

  • ✅ Ideal para principiantes por su sintaxis clara
  • ✅ Bibliotecas como Pandas, NumPy, TensorFlow
  • ✅ Usado en IA, análisis de datos, web, scripts

☕ Java

Java es un lenguaje orientado a objetos, muy usado en el desarrollo de aplicaciones empresariales, móviles y sistemas robustos.

  • ✅ Multiplataforma gracias a la JVM
  • ✅ Amplio uso en empresas y backend
  • ✅ Android, Spring Boot, arquitectura sólida
